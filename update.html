MEAN Stack

MongoDB, ExpressJS, AngularJS, NodeJS

A bit more advanced but provides more standardization and cleaner/more coherent data transferal.


LAMP Stack

Linux, Apache, MySQL, PHP (or Perl if using Python)

A bit more beginner friendly and allows for a greater range of customization between user and end points of an app.


Between LAMP and MEAN stacks, I'd prefer working with the MEAN stack. More particularly, the MERN Stack, which swaps out Mongo for Mongoose and AngularJS for ReactJS.
